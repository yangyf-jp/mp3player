# MP3 Player - Flutter Windows 应用

一个美观大方的 MP3 播放器，支持播放列表管理和多种播放策略。

## 功能特性

### 1. 播放列表功能

- **新建播放列表**: 可以创建自定义名称的播放列表，支持添加备注
- **拖拽添加文件**: 
  - 直接拖拽多个 MP3 文件到播放列表
  - 拖拽文件夹时自动将文件夹中的所有 MP3 文件添加到播放列表
- **拖拽排序**: 可以通过拖拽调整播放列表中歌曲的顺序
- **多播放列表管理**: 
  - 支持创建多个播放列表
  - 可以删除不需要的播放列表
  - 可以修改播放列表属性（名称、备注等）
- **特殊播放列表**: 可以标记某些播放列表为"特殊播放列表"，方便快速添加歌曲
- **数据库管理**: 所有播放列表数据存储在 SQLite 数据库中
- **播放策略切换**: 每个播放列表可以独立设置播放策略，通过下拉列表快速切换
- **即时播放**: 在播放列表界面直接播放 MP3，实时显示当前播放的歌曲
- **动态策略生效**: 播放过程中修改播放策略，在下一曲播放前生效
- **快速添加到特殊列表**: 点击歌曲后的 + 按钮，弹出特殊播放列表选择框

### 2. 播放策略功能

- **策略管理**: 支持播放策略的新增、修改和删除
- **策略配置**:
  - **播放间隔**: 一曲播完后等待指定秒数再播放下一曲
  - **单曲重复次数**: 每首歌曲重复播放的次数
  - **播放模式**: 
    - 顺序播放：按列表顺序依次播放
    - 随机播放：随机选择下一首歌曲
  - **播放控制**:
    - 单次播放：播放完整个列表后停止
    - 循环播放：播放完整个列表后重新开始
- **使用保护**: 正在被播放列表使用的策略不可删除，但可以修改
- **数据库存储**: 所有播放策略存储在 SQLite 数据库中

## 技术架构

### 项目结构

```
lib/
├── main.dart                 # 应用入口
├── models/                   # 数据模型
│   ├── playlist.dart         # 播放列表和曲目模型
│   └── playback_strategy.dart # 播放策略模型
├── providers/                # 状态管理
│   └── app_provider.dart     # 应用状态提供者
├── services/                 # 服务层
│   ├── database_service.dart # 数据库服务
│   └── audio_player_service.dart # 音频播放服务
├── screens/                  # 页面
│   └── home_screen.dart      # 主页面
└── widgets/                  # 可复用组件
    ├── playlist_list.dart    # 播放列表组件
    ├── playlist_detail.dart  # 播放列表详情组件
    ├── player_controls.dart  # 播放控制栏组件
    └── strategy_manager.dart # 播放策略管理组件
```

### 依赖库

- **provider**: 状态管理
- **sqflite_common_ffi**: SQLite 数据库支持（Windows 平台）
- **just_audio**: 音频播放
- **file_picker**: 文件选择
- **uuid**: 唯一标识符生成
- **google_fonts**: 字体美化

## 运行要求

- Flutter SDK >= 3.0.0
- Windows 10/11
- 已配置 Flutter Windows 桌面开发环境

## 安装和运行

1. 克隆或下载项目到本地

2. 安装依赖
   ```bash
   flutter pub get
   ```

3. 运行应用
   ```bash
   flutter run -d windows
   ```

4. 构建发布版本
   ```bash
   flutter build windows --release
   ```

## 使用说明

### 创建播放列表

1. 点击左侧边栏的 "+" 按钮
2. 输入播放列表名称和备注（可选）
3. 可选择是否为"特殊播放列表"
4. 点击"创建"

### 添加歌曲

**方法一：拖拽添加**
- 直接从文件资源管理器拖拽 MP3 文件或文件夹到播放列表区域

**方法二：从其他播放列表添加**
- 点击歌曲后面的 "+" 按钮
- 选择要添加到的特殊播放列表

### 播放控制

- 点击播放列表顶部的播放按钮开始播放
- 点击任意歌曲立即播放该歌曲
- 使用底部控制栏进行播放/暂停、上一曲、下一曲操作

### 设置播放策略

1. 在播放列表顶部选择播放策略下拉框
2. 或在左侧导航栏选择"播放策略"进行详细配置

### 管理播放策略

1. 点击左侧导航栏的"播放策略"
2. 点击"新建策略"创建新策略
3. 配置策略参数：
   - 策略名称
   - 播放间隔（秒）
   - 单曲重复次数
   - 播放模式（顺序/随机）
   - 播放控制（单次/循环）
4. 编辑或删除已有策略（正在使用的策略只能编辑不能删除）

## 界面设计

应用采用深色主题设计，主要特点：

- **左侧导航栏**: 播放列表和播放策略管理入口
- **主内容区**: 显示当前选中的播放列表或策略管理界面
- **底部控制栏**: 显示当前播放信息和播放控制按钮
- **美观的渐变色**: 播放列表头部采用蓝色渐变背景
- **清晰的视觉反馈**: 当前播放的歌曲高亮显示，带动态均衡器图标

## 数据存储

应用使用 SQLite 数据库存储：

- 播放列表信息（名称、备注、策略关联等）
- 播放列表中的曲目信息
- 播放策略配置

数据库文件位置：`%APPDATA%/mp3_player.db`

## 注意事项

1. 首次运行时会自动创建默认播放策略
2. 删除播放列表会同时删除其中的所有曲目记录
3. 正在使用的播放策略无法删除
4. 拖拽文件夹时只识别 .mp3 扩展名的文件
